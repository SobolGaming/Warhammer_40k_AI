# Decision Submission Catalog

This document is the authoritative catalog of the adapter-visible decision submission
surfaces currently exposed by CORE V2. Finite rows are answered with
`FiniteOptionSubmission`. Parameterized rows are answered with `ParameterizedSubmission`
using the fixed option ID `submit_parameterized_payload`.

The catalog lists concrete `decision_type` tokens and concrete proposal kinds so UI,
headless, network, replay, and test adapters can bind to the same lifecycle path.
